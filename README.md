# Проект API_Parser_Dj
## API_Parser_Dj — это вебсайт на Django для парсинга вакансий с сайтов и анализа данных. Проект предоставляет возможность поиска вакансий по ключевым словам и регионам, отображения статистики по навыкам, а также включает блог для обсуждения вакансий и карьеры. Кроме того, реализован личный кабинет пользователя и раздел FAQ.

# Основные функции

**Парсер вакансий:**

Поиск вакансий по ключевым словам и регионам.

Отображение списка вакансий с пагинацией.

Топ-10 навыков, которые упомянуты в вакансиях.

**Блог:**

Пагинация постов.

Поиск постов по тегам.

Создание, редактирование и удаление постов (для авторов).

Комментирование постов (для зарегистрированных пользователей).

**Личный кабинет пользователя:**

Управление аватаром.

Запрос статуса "Автор" для публикации постов.

Получение уведомлений от администратора.

**FAQ:**

Раздел с часто задаваемыми вопросами.

Управление вопросами и ответами через админку.

# Технологии

Python 3.12 — язык программирования.

Django 5.1 — веб-фреймворк для создания приложения.

SQLite — база данных по умолчанию (или другая, если используется).

Дополнительные библиотеки — укажи, если используются (например, для парсинга или анализа данных).

# Установка и настройка
1. Установите Python 3.12, если он ещё не установлен.

2. Клонируйте репозиторий:

git clone https://github.com/ваш-репозиторий.git
cd ваш-репозиторий

3. Создайте виртуальное окружение:

python -m venv venv
source venv/bin/activate  # для Linux/MacOS
venv\Scripts\activate     # для Windows

4. Установите зависимости:

pip install -r requirements.txt

5. Настройте базу данных в settings.py:

DATABASES = {
    'default': {
        'ENGINE': 'django.db.backends.sqlite3',
        'NAME': BASE_DIR / 'db.sqlite3',
    }
}

6. Выполните миграции:

python manage.py migrate

7. Создайте суперпользователя:

python manage.py createsuperuser

8. Откройте браузер и перейдите по адресу:

http://127.0.0.1:8000/

**Форма поиска вакансий**

Форма поиска вакансий реализована в файле parserapp/forms.py и позволяет пользователям искать вакансии по различным параметрам. Вот основные поля формы:

Поля формы:
Название вакансии (title):

Поле для ввода названия вакансии.

Пример: "Python разработчик".

Виджет: текстовое поле с подсказкой.

Местоположение (location):

Поле для указания города или страны, где искать вакансии.

Опциональное поле.

Виджет: текстовое поле с подсказкой.

Опыт работы (experience):

Выпадающий список для выбора опыта работы.

Варианты:

Без опыта.

От 1 до 3 лет.

От 3 до 6 лет.

Более 6 лет.

Виджет: выпадающий список.

Формат работы (work_format):

Выпадающий список для выбора формата работы.

Опциональное поле.

Варианты:

Удаленно.

Гибрид.

Офис.

Виджет: выпадающий список.

Тип занятости (employment_type):

Выпадающий список для выбора типа занятости.

Опциональное поле.

Варианты:

Полный день.

Частичная занятость.

Гибкий график.

Виджет: выпадающий список.

**Отображение результатов поиска**

Результаты поиска вакансий отображаются на отдельной странице с пагинацией. Вот основные особенности:

Список вакансий:

Каждая вакансия отображается в виде карточки с краткой информацией:

Название вакансии.

Компания.

Местоположение.

Зарплата (если указана).

Ссылка на подробное описание.

Пагинация:

Если результатов больше, чем задано на одной странице, они разбиваются на несколько страниц.

Пользователь может переключаться между страницами с помощью пагинации внизу списка.

Топ-10 навыков:

На странице результатов поиска отображается статистика по навыкам.

Пользователь видит топ-10 навыков, которые чаще всего упоминаются в вакансиях.
